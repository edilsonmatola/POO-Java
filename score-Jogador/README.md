# Player Score

Programa de criacao de um jogador, na qual exibe uma tela, onde deve preencher o nome do
jogador e colocar o seu devido score. De seguida, podera aceder aos dados inseridos do jogador 
atraves da **opcoes** que localiza-se na barra de menu da tela onde ira aceder atraves da opcao exibir. 
O programa ira criar um ficheiro txt onde sera guardados os dados introduzidos na tela.
